# Welcome to Bioinformatics Basics

This is a site I made to help introduce people to bioinformatics basics, most of it is adapted from BME110 (Introduction to Bioinformatics Tools) taught by Todd Lowe at UCSC. I will add to it as we progress and add any useful resources we find.
{: .fs-6 .fw-300 }

# Resources

## Linux/UNIX Terminal

* [Code Snipcademy](https://code.snipcademy.com/tutorials/linux-command-line){:target="blank"}: Great online resource for basic / intermediate Linux
* [UC Davis ECS 40](https://csiflabs.cs.ucdavis.edu/~ssdavis/40/UNIX%20Lecture%20Notes%20(BASH).pdf){:target="blank"}: Great set of lecture notes for learning
  * Great for `wc`,`grep`,`sftp`, and `vim`
* [Sean Davis's Tutorial](https://csiflabs.cs.ucdavis.edu/~ssdavis/30/UnixWorkshop.pdf){:target="blank"}
* [Vince Buffalo](https://vincebuffalo.com/blog/2013/08/08/using-names-pipes-and-process-substitution-in-bioinformatics.html){:target="blank"}: Advanced substitution and piping

## Python

* [Pandas Getting Started](https://pandas.pydata.org/pandas-docs/stable/getting_started/10min.html){:target="blank"}: Basic dataframes in python
* [Rosalind](http://rosalind.info/problems/locations/){:target="blank"}: Bionformatics algorithm basics via python
* [Colab Notebooks](https://towardsdatascience.com/getting-started-with-google-colab-f2fff97f594c?gi=88c04ac04035){:target="blank"}: Less flexiable than Jupyter notebooks but can be deployed quickly without having to install software, and run through google drive
* [Jupyter Notebooks](https://www.datacamp.com/community/tutorials/tutorial-jupyter-notebook){:target="blank"}: Great alternative to plain python for data manipulation, plot generation, machine learning and data exploration
