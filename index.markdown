# Welcome to Bioinformatics Basics

This is a site I made to help introduce people to bioinformatics basics, most of it is adapted from UCSC BME110 taught by Todd Lowe. I will add to it as we progress and add any useful resources we find.

# Resources

## Linux/UNIX Terminal

* [Code Snipcademy](https://code.snipcademy.com/tutorials/linux-command-line): Great online resource for basic / intermediate Linux
* [UC Davis ECS 40](https://csiflabs.cs.ucdavis.edu/~ssdavis/40/UNIX%20Lecture%20Notes%20(BASH).pdf): Great set of lecture notes for learning
  * Great for `wc`,`grep`,`sftp`, and `vim`
* [Sean Davis's Tutorial](https://csiflabs.cs.ucdavis.edu/~ssdavis/30/UnixWorkshop.pdf)
* [Vince Buffalo](https://vincebuffalo.com/blog/2013/08/08/using-names-pipes-and-process-substitution-in-bioinformatics.html): Advanced substitution and piping

## Python

* [Pandas Getting Started](https://pandas.pydata.org/pandas-docs/stable/getting_started/10min.html): Basic dataframes in python
* [Rosalind](http://rosalind.info/problems/locations/): Bionformatics algorithm basics via python
